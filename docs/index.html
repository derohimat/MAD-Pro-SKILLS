MAD Pro CLI

Elevate your Android projects with a comprehensive library of 35+ professional MAD skills, optimized for AI-driven development.

$ npm install -g mad-pro-cli

Quick Start

1. Install Global

Install the CLI tool globally to access the mad-pro command anywhere.

npm i -g mad-pro-cli

2. Initialize Skills

In your project directory, run the init command to bootstrap your AI skills.

mad-pro init --ai antigravity

3. Boost AI

Your AI agent (like Antigravity) will now find expert guides in .agent/skills/.
