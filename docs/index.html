How It Works

1

Bootstrap or Init

Start fresh with mad-pro create to clone our KrossWave KMP template, or run mad-pro init in an existing project to start the injection wizard.

$mad-pro create my-awesome-app
2

Smart Selection

Our interactive wizard helps you pick your IDE and select the specific skill categories (Core, Industry, AI) you need. No token bloat, just the context you need.

$mad-pro init
3

Maintain & Audit

Use mad-pro doctor to audit your architecture or mad-pro add to fetch new skills as your project evolves.

$mad-pro doctor
4

Agent Empowerment

Your AI agent automatically reads the local skill context and starts producing architecture-compliant, production-grade Android code via optimized .cursorrules.

MAD Pro Doctor

The Ultimate Architecture & Security Scanner

Adaptive scanning tailored to your team's chosen architecture, plus deep security insights and PR reviews.

🏛️ Architecture Templates

Supports scanning customized to MVVM, MVI, MVP, Clean+MVI, or VIPER. Automatically detects DI usage (Hilt/Koin).

🔒 Security & Leaks

Scans for hardcoded APIs, HTTP usage, unencrypted prefs, unprotected broadcast receivers, and GlobalScope leaks.

Sub-modules & Flags:

mad-pro doctor

Default run. Scans your entire codebase recursively for architecture anti-patterns, DI bypasses, and God-classes.

--security

Stacks a deep security scan. Detects hardcoded API keys, insecure HTTP endpoints, unencrypted prefs, and GlobalScope leaks.

--review <branch>

Performs a targeted scan utilizing git diff. Exclusively analyzes files changed against the target branch. Excellent for PR validations.

--fix Beta

Automatically scaffolds required classes (e.g. GeneratedUseCase.kt) and provides drop-in code refactors straight to your console.
