A comprehensive library of **35+ professional MAD skills** curated for Modern Android Development. These skills are designed to be ingested by AI coding agents (like Antigravity) to ensure your projects follow industry-best practices from day one.

## 🏗️ What's Inside?

This repository contains expert reference guides, architectural patterns, and engineering principles categorized by the MAD layers:

- **🎨 UI Layer**: Jetpack Compose, Material 3, Adaptive Layouts, Motion, and Accessibility.
- **🧠 Domain Layer**: Use Cases, Clean Architecture, and SOLID Principles.
- **💾 Data Layer**: Room, DataStore, Ktor, and Offline-first strategies.
- **🛠️ Engineering Excellence**: CI/CD (Fastlane/GitHub Actions), Design Systems, and On-Device AI (Gemini Nano).
- **🛤️ Migration**: Standardized roadmaps for XML to Compose and Android to KMP/CMP.

## 🚀 Quick Start (CLI)

The easiest way to integrate these skills into your project is using our official CLI tool:

### 1. Install globally
```bash
npm install -g mad-pro-cli
```

### 2. Initialize in your project
```bash
mad-pro init --ai antigravity
```
*This will install the skills into your project's `.agent/skills/mad-skills/` directory.*
